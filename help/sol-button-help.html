sol-button

A button that opens something when you click it — a page, a file, another component, or an outside website. You choose what it opens and where it appears: in an area of your page, or in a pop-up. Think of it as a link you can point anywhere.

Try it

Use it in your HTML

Add this line once, near the top of your page — it makes the components available:

<script src="dist/sol-loader.min.js" data-bundles="sol-basic"></script>

Then use the tag anywhere you like. The words between the tags are the button's label:

<sol-button source="welcome.html" region="#main">Welcome</sol-button>

That makes a button labelled Welcome; clicking it loads welcome.html into the element on your page with id="main".

Options

AttributeWhat it does
source What to open — a page, a file, or a web address.
region Where it opens. Either a CSS selector for an area of your page (for example #main), or one of the pop-up keywords modal (a centered dialog) or floating (a draggable window).
handler Which component opens the content. You can usually leave this off — same-site pages open inline, outside websites open in a frame. Set it (e.g. sol-query) only when you want a specific one.
name A short label for the content, used when it opens as a tab.
replace Reload the content fresh on every click. By default it's kept between clicks so it opens instantly the second time.

Examples

<!-- open a page in the #main area -->
<sol-button source="pages/settings.html" region="#main">Settings</sol-button>

<!-- pop up a dialog -->
<sol-button source="data/reports.ttl" region="modal">Reports</sol-button>

<!-- open an outside website (shows up in a frame automatically) -->
<sol-button source="https://solidproject.org" region="#main">Website</sol-button>

<!-- several buttons that all open into the same area -->
<div data-region="#aux">
  <sol-button source="a.html">A</sol-button>
  <sol-button source="b.html">B</sol-button>
</div>

Tip: instead of repeating region on every button, put it once on a wrapping element with data-region="…" and the buttons inside will use it.
