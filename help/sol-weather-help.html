sol-weather

A small current-conditions card — icon, place name, temperature, and the highest chance of rain over the next few hours. The data comes from the free Open-Meteo service (no API key, no proxy needed) and refreshes every 10 minutes.

Try it

Use it in your HTML

Add this line once, near the top of your page — it makes the components available:

<script src="https://cdn.jsdelivr.net/npm/component-interop@0.2.1/component-interop.js"
        data-manifest="../dist/sol-components.manifest.json"
        data-components="sol-include sol-weather"></script>

Then point the tag at a small settings file that says which place to show:

<sol-weather source="data/weather-settings.ttl#Settings"></sol-weather>

Options

sol-weather is configured from a small settings file rather than from many attributes. The settings inside that file:

SettingWhat it does
geo:lat Latitude (−90 to 90). Required.
geo:long Longitude (−180 to 180). Required.
schema:addressLocality The place name shown on the card.
ui:temperatureUnit Which units to show: ui:Fahrenheit, ui:Celsius, or ui:Both.
time:hours How many hours ahead to check for the highest rain chance (default 12).

Some attributes here are shared across components — see Common attributes.
