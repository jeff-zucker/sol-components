sol-pod

A file browser for a user's Solid pod (their personal online storage). It shows a pod's folders and files, lets the user switch between pods, and tells your page which item they clicked so you can react to it.

Try it

Use it in your HTML

Add this line once, near the top of your page — it makes the components available:

<script src="https://cdn.jsdelivr.net/npm/component-interop@0.2.1/component-interop.js"
        data-manifest="../dist/sol-components.manifest.json"
        data-components="sol-pod"></script>

Then drop the tag in. With no source it finds the pods for the signed-in user on its own:

<sol-pod></sol-pod>

Options

AttributeWhat it does
source The pod address to show, or a comma- or space-separated list of several. Leave it off to find the signed-in user's pods automatically.
gear-icon Replaces the default ⚙ button next to each item. Give it a short bit of text (like a single emoji) to use as text, or an image address (one containing a / or ending in .svg/.png/.jpg/.gif/.webp) to show a picture.
pod-click-action What happens when the user picks an item (clicks its icon, presses Enter, or double-clicks). Set it in JavaScript to a function; it receives the item, including its url, name, displayName, whether it's a folder, and its content type. handler is another name for the same thing.
login-mode, login-callback, issuers, side Passed straight through to the built-in sign-in box, to control how and where signing in happens.
pods-group Lets several <sol-pod> tags on one page share the same pod list. Leave it off to share by default, or set it to none to keep a tag on its own.

Some attributes here are shared across components — see Common attributes.

Examples

<!-- find the signed-in user's pods automatically -->
<sol-pod></sol-pod>

<!-- show one specific pod -->
<sol-pod source="https://alice.solidcommunity.net/"></sol-pod>

<!-- use a different icon next to each item -->
<sol-pod gear-icon="📁"></sol-pod>

To react when the user picks a file, set podClickAction in JavaScript — that's what the live demo above does, showing the item's details in the box under the browser:

<script type="module">
  await SolidWebComponents.ready;
  const pod = document.querySelector('sol-pod');
  pod.podClickAction = (item) => {
    console.log('picked', item.name, item.url);
  };
</script>
