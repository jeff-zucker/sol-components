sol-time

A compact clock. It always shows your local time and GMT, and can add a third clock for any timezone you name. It updates once a minute (no ticking seconds).

Try it

Use it in your HTML

Add this line once, near the top of your page — it makes the components available:

<script src="dist/sol-loader.min.js" data-bundles="sol-include sol-time"></script>

Then point the tag at a small settings file that says which extra timezone to show:

<sol-time source="data/time-settings.ttl#Settings"></sol-time>

Options

sol-time is configured from a small settings file rather than from many attributes. The one setting inside that file:

SettingWhat it does
schema:timezone The extra timezone to show, given as an IANA timezone name such as "Asia/Kolkata" or "America/New_York". The label shown is the last part of the name (Kolkata), and the time is worked out from that zone automatically — there's no separate offset to set.

Some attributes here are shared across components — see Common attributes.
