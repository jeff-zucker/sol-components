SPARQL on a plain element — data-from-query

A plain <div> made query-driven: the sparql capability's activator reads the element's data-from-query + the usual sol-query attributes (endpoint, view, …), runs the query, and replaces the element's content. HTML views render in place; a URL view (view="…/view.js") gets the result as a W3C SPARQL 1.1 Query Results JSON object via render(container, data, el).
