sol-solidos

Embeds the SolidOS data browser right in your page. Point it at a Solid resource with the source attribute and it shows that resource in the SolidOS browser, with all its panes for viewing and editing the data. Use it when you want a full data browser inside your own page instead of building one.

Use it in your HTML

Add this line once, near the top of your page — it makes the components available:

<script src="dist/sol-loader.min.js" data-with="solidos"></script>

Then drop the tag in and give it a resource to show:

<sol-solidos source="https://your.pod/some/resource"></sol-solidos>

Options

AttributeWhat it does
source The Solid resource to display. Change it after the page has loaded and the browser navigates to the new resource. Leave it off and it starts on the current page's address.

source is shared across components — see Common attributes.

Examples

<!-- browse one resource -->
<sol-solidos source="https://your.pod/notes/today.ttl"></sol-solidos>

<!-- start on whatever page the user is on -->
<sol-solidos></sol-solidos>

Changing source later — for example from a button or a menu elsewhere on the page — moves the browser to that resource.
