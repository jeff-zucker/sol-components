sol-query

Query SPARQL endpoints or plain RDF files using Comunica or triple patterns. Stored queries, replaceable parameters, and multi-endpoint federation are all supported.

Query Component HTML (table) 
<sol-query
    endpoint="../data/sample-data.ttl"
    sparql="PREFIX schema: <http://schema.org/>
            SELECT ?name ?gender ?location WHERE {
              ?p a schema:Person ;
                 schema:name ?name ;
                 schema:gender ?gender ;
                 schema:homeLocation ?location .
            }
    view="table"
></sol-query>
