sol-menu component

A navigation strip plus a content panel. Fill it with <a href> children — each becomes a menu entry; clicking one loads its href into the panel. Group entries under collapsible <submenu> sections to any depth. The first entry is selected on load so the panel is never empty.

Authoring

Entries are anchors, exactly like <sol-tabs>: href is the content URL, the link text is the label, and content mounts with sol-include unless you set a handler (per anchor, or on <sol-menu> as a default). A <submenu> holds a <label> heading followed by its own entries (or further <submenu>s) and opens as a fly-out.

An entry can be marked if-logged-in (or requires-write) to flag that it needs an authenticated, writable session. The menu takes no policy itself — it exposes such entries as ::part(requires-write) so the host page decides whether to hide or disable them.

Attributes

AttributePurpose
handler Default component tag for every entry (an anchor's own handler overrides it). Falls back to sol-include.
orientation horizontal lays the nav bar on top; default is a sidebar on the left.
icon (on an entry) Show an icon (image URL or inline SVG data-URI) instead of text; the label becomes the tooltip / aria-label.
if-logged-in (on an entry) Flag the entry as needing a writable session; surfaced via ::part(requires-write) for the host to gate.

The content panel is a light-DOM .sol-menu-content child, so results are reachable by ordinary page CSS / queries. Selecting an entry fires sol-menu-change (detail: { name }).

Examples

<!-- sidebar with a nested group -->
<sol-menu>
  <a href="home.html">Home</a>
  <submenu>
    <label>Docs</label>
    <a href="quickstart.md">Quickstart</a>
    <submenu>
      <label>API</label>
      <a href="api/query.md">Query</a>
      <a href="api/modal.md">Modal</a>
    </submenu>
  </submenu>
</sol-menu>

<!-- a top nav bar with a default handler -->
<sol-menu orientation="horizontal" handler="sol-live-edit">
  <a href="readme.md">Readme</a>
  <a href="changelog.md">Changelog</a>
</sol-menu>
