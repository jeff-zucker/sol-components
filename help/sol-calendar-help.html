sol-calendar

Shows an agenda of upcoming events on your page. Point it at one or more public calendar (ICS) feeds — Google, Apple, Outlook, Proton, or a Solid pod — and it merges them into a single, sorted list.

Try it

Use it in your HTML

Add this line once, near the top of your page — it makes the component available:

<script src="https://cdn.jsdelivr.net/npm/component-interop@0.2.2/component-interop.js"
        data-manifest="../dist/sol-components.manifest.json"
        data-components="sol-include sol-calendar"></script>

Then point the tag at a small settings file (more on that below) with a #fragment picking which record to use:

<sol-calendar source="data/calendar-settings.ttl#All"></sol-calendar>

Options

Some attributes here are shared across components — see Common attributes. The two that matter most: source points at the settings file (with a #fragment), and proxy is a CORS proxy address the feeds are read through. Most providers can't be read directly by a browser, so you'll usually need proxy — leave it off only for a feed that a browser is already allowed to read, like a Solid pod or a self-hosted file. An HTML attribute of the same name always overrides the value in the settings file.

Examples

Point source at a small settings file. Each record lists its calendar feeds and a few display choices. Here's the file the demo above uses:

<sol-calendar source="data/calendar-settings.ttl#All"></sol-calendar>

data/calendar-settings.ttl:

The settings a record can carry:

SettingWhat it does
dct:source A calendar feed's web address. Repeat it to merge several feeds into one agenda.
dct:format "ics" or "google" — which kind of feed it is.
ui:view "agenda" — the list layout (month / week / day are planned).
time:days How many days ahead to look.
schema:numberOfItems An optional cap on how many events to show. (The schema.org namespace is always http://schema.org/, never the HTTPS form.)

Finding a calendar feed address

The feed address must be a calendar export (it returns text/calendar and starts with BEGIN:VCALENDAR) — not the address of the calendar page you see in your browser, which returns a web page and fails. Where to find the export link:

  • Google — Settings and sharing → "Integrate calendar". The secret address in iCal format works for any calendar (treat that link like a password). The public address works once the calendar is made public.
  • Apple iCloud — right-click the calendar → Share → Public Calendar; swap webcal:// for https://.
  • Outlook — Settings → Calendar → Shared calendars → Publish a calendar; copy the ICS link.
  • Proton — calendar settings → Share publicly → "Subscribe with calendar app".
  • Self-hosted / Solid pod — any address that returns text/calendar.

Meeting links

When an event has a video-meeting link (Jitsi, Zoom, Google Meet, Teams, Whereby), the event title becomes a link to it, so the agenda doubles as a one-click way to join.
