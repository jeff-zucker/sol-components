Theming & customization

You theme these components with plain CSS — no special setup. They read their colors, fonts, and spacing from CSS custom properties (the --name variables you may already use). Set one of those variables on your page and every component picks up your value.

How it works

Set a variable at the top of your page, in :root, and the components see it:

:root {
  --accent: #ff6f00;          /* the link / focus / hover color, everywhere */
  --medium-font: 18px;        /* the base font size */
}

You can also change a variable for just one part of the page — set it on a specific element and only the components inside that element are affected:

aside.sidebar {
  --menu-active-bg: #fff5e6;  /* only the menu inside this sidebar changes */
}

Some components have their own variables (named --menu-*, --query-*, --include-*). When one of those is set it wins over the matching general variable, so you can restyle a single component without affecting the others.

Light & dark themes

Light is the default. To switch to the built-in dark theme, add data-theme="dark" to your <html> tag (or any element above your components):

<html data-theme="dark">
  ...
</html>

The dark theme just gives these variables new values, and every component that uses them switches over:

VariableLight defaultDark value
--bg #f5f5f5#1a1a1a
--surface #ffffff#252525
--border #d0d0d0#3e3e3e
--text #000000#e0e0e0
--text-muted #4d4d4d#909090
--hover #eaf2fb#2e2e2e
--focus-bg #ebf5fb#2a2e34
--code-bg #f4f4f4#1e1e1e
--accent #1F618D#4dabf7
--accent-dark #2980b9#339af0

To follow the visitor's system setting automatically:

<script>
  if (matchMedia('(prefers-color-scheme: dark)').matches) {
    document.documentElement.dataset.theme = 'dark';
  }
</script>

To make your own theme, set just the variables you want to change — you don't have to list them all:

:root[data-theme="brand"] {
  --accent:      #ff6f00;
  --accent-dark: #c44a00;
  --hover:       #fff3e0;
  --focus-bg:    #ffe0b2;
}

Typography

There are three sizes — small, medium, large — plus a --font-size that means "the default body size." Medium is the default, and nothing renders below 16px.

VariableDefaultWhat it controls
--small-font 16pxCompact text — field labels, sort markers, button text in tight spots, badges
--medium-font 20pxDefault body / control / table size
--large-font 24pxSection headers (e.g. sol-query subject banner)
--font-size var(--medium-font, 20px)The default size components reach for first; change it to retag the base size
--font-ui 'Segoe UI', Roboto, sans-serifUI font family
--font-mono 'Cascadia Code', 'Fira Code', 'Consolas', monospaceMonospace family for code blocks and Turtle output
--font-weight-normal400Body weight
--font-weight-bold 600Bold weight (table headers, active items, field labels)
--line-height-tight 1.25Headings
--line-height-base 1.5Body text
To make everything smaller (or bigger) at once: change --medium-font (or --font-size). Text that uses --small-font or --large-font keeps its own tier — small stays smaller than medium even after you change medium.

Recipes

Change the brand color (one place, affects everything)

:root {
  --accent:      #ff6f00;
  --accent-dark: #c44a00;
}

Compact mode (smaller text, tighter spacing)

:root {
  --medium-font: 16px;
  --small-font:  14px;   /* below 16px — fine if you accept the smaller size */
  --space-md:    6px;
  --space-lg:    8px;
}

Theme just the menu, leave the rest alone

:root {
  --menu-active-bg:    #fff3e0;
  --menu-active-color: #c44a00;
  --menu-hover-bg:     #fff8e1;
}

Flat table instead of stripes

:root {
  --query-row-alt-bg: var(--surface);   /* even rows match odd rows */
}

Auto dark mode (follow the OS setting)

<script>
  const root = document.documentElement;
  const apply = e => root.dataset.theme = e.matches ? 'dark' : 'light';
  const mql = matchMedia('(prefers-color-scheme: dark)');
  apply(mql); mql.addEventListener('change', apply);
</script>

A few themes the visitor can switch between

:root[data-theme="ocean"]  { --accent: #0277bd; --accent-dark: #01579b; --hover: #e1f5fe; }
:root[data-theme="forest"] { --accent: #2e7d32; --accent-dark: #1b5e20; --hover: #e8f5e9; }
:root[data-theme="sunset"] { --accent: #e64a19; --accent-dark: #bf360c; --hover: #fff3e0; }
// switch in JavaScript
document.documentElement.dataset.theme = 'forest';
